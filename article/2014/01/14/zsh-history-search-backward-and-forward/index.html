ZSH - history-search-backward/forward

2014-01-14
Note
Ubuntu
ZSH
0 Comment

原本oh-my-zsh在Ubuntu 12.04 LTS以及Arch Linux用好好的history-search-backward、history-search-forward，到了Ubuntu 13.10就不能用了，查了一下ZSH雖然版號不一樣，不知道是不是因為發行版影響，因為Arch Linux的ZSH版本還比Ubuntu 13.10新。

下面是各個發行版的Z Shell版號。

Ubuntu 12.04 LTS : 4.3.17
Ubuntu 12.10     : 5.0.0
Ubuntu 13.04     : 5.0.0
Ubuntu 13.10     : 5.0.2
Arch Linux       : 5.0.5

算了至少功能沒有那就自己加上去，在~/.zshrc中加入以下兩行就行了。

bindkey "${key[Up]}" history-search-backward
bindkey "${key[Down]}" history-search-forward

接著重起terminal或是source .zshrc一切就完成了。
